    About the project

    A custom-built Wordle clone featuring the classic 6-guess word game mechanic. Score accumulates across rounds — 60 points for a first-guess ace, dropping 10 points for each extra attempt. Guess wrong on guess 6 and the run ends, giving you a shot at the leaderboard.

    Built entirely from scratch using vanilla JavaScript — no libraries or frameworks. The word list is loaded from a local text file and shuffled each session so the order is different every time you play.

    What I built

    • Full Wordle game logic — correct / present / absent evaluation with proper duplicate letter handling
    • On-screen keyboard with colour feedback, plus full physical keyboard support
    • Tile flip reveal animation and row shake on invalid word
    • Score system across consecutive rounds with a live leaderboard
    • Leaderboard persisted via Cloudflare KV — scores survive page reloads
