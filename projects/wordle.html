Wordle 🟩

A fully playable Wordle clone with score tracking and a leaderboard, built from scratch in JavaScript.

About the project

A custom-built Wordle clone featuring the classic 6-guess word game mechanic. The game tracks your score across rounds and includes a live leaderboard so you can compete with others.

Built entirely from scratch using vanilla JavaScript — no libraries or frameworks. A great exercise in DOM manipulation, game state management, and keyboard event handling.

What I learned

  • Managing complex game state in vanilla JS
  • Keyboard input handling and on-screen keyboard sync
  • Building a leaderboard with persistent scoring
  • CSS animations for tile reveals and keyboard feedback
