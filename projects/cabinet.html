Cabinet 🗄️

A 3D modelled cabinet with an interactive web demo for embedding and displaying 3D models in the browser.

About the project

A focused experiment combining 3D modelling with web development, a cabinet modelled in detail and then embedded interactively on a webpage. The goal was to explore how 3D models can be displayed natively in the browser, letting users rotate and inspect the object without needing any special software.

This was an early exploration into the kind of 3D product visualisation I used later in projects like Mockford Blinds.

What I learned

  • 3D model export formats for web (e.g. glTF, OBJ)
  • Embedding interactive 3D models in a webpage
  • Browser-based 3D rendering considerations
