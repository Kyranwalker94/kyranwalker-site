AMD Desktop PC Build 🔴

My current daily driver — an AM4 platform build put together in 2020 and upgraded since, balancing performance, value, and longevity.

About the build

After years on Intel, I switched to AMD's AM4 platform in 2020. The Ryzen lineup was competitive in ways Intel hadn't faced in years, and AM4's longevity — multiple CPU generations on the same socket — made it an attractive long-term platform to invest in.

This is my current main machine, used for everything from gaming and 3D printing workflows to web development and general productivity. It's been upgraded incrementally since the initial build and remains well within its useful life.

Why AM4?

AM4 was compelling in 2020 for a few reasons: AMD's Ryzen 3000 and 5000 series were genuinely beating Intel on both performance and price per core, and the platform had already demonstrated support for multiple CPU generations — meaning a future CPU upgrade wouldn't require a motherboard replacement.

That longevity has proven out. The machine has had incremental component upgrades without a full rebuild, which is exactly what you want from a platform choice.

Use cases

  • Gaming — primary platform for PC gaming
  • 3D print workflow — slicing, file prep, and OctoPrint management for the Ender 3 V2 setup
  • Web development — local development environment, browser testing
  • 3D modelling — SketchUp, Blender, and SOLIDWORKS projects
  • General productivity — spreadsheets, documents, and data analysis
