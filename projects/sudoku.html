Sudoku 🔢

A fully playable Sudoku with four difficulty levels, pencil notes, conflict highlighting, and a timer — built from scratch in vanilla JS.

🔢

Game in development

The Sudoku game is being built. Check back soon — it'll support Easy, Medium, Hard, and Expert puzzles with notes mode and conflict detection.

How it will work

The game uses a pre-generated bank of puzzles for each difficulty level, so there's no puzzle generation delay on load. Every puzzle is guaranteed to have exactly one solution.

  • Select a cell, then press a number key or use the on-screen numpad
  • Pencil mode — toggle notes on/off to track candidates in any cell
  • Conflict highlighting — rule violations are flagged immediately without removing your entry
  • Row, column, and box highlighting when a cell is selected
  • Same-digit highlighting — selecting any filled digit highlights all matching numbers on the board
  • Timer running from first input
  • Check button — marks all current errors
