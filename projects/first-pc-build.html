First Desktop PC Build 🖥️

My first ever custom desktop build — assembled from scratch in 2012 inside an NZXT Phantom case, centred on an Intel i5 and a then-current Nvidia GPU.

About the build

Before this build I'd only ever used laptops or family computers. Building a PC from scratch in 2012 was a proper rite of passage — researching every component, watching hours of build guides, and crossing my fingers when it powered on for the first time.

The NZXT Phantom was a bold choice — a massive full tower with angular lines and LED lighting. It was partly impractical (the thing was enormous) and entirely the right call for a first build. Plenty of space, great cable management, and unmistakably a gaming PC.

Storage strategy

At the time, SSDs were still expensive per gigabyte, so the setup was a classic two-drive split: the 90 GB Corsair SSD for the OS and primary applications, and the 1 TB Seagate spinning disk for games, files, and everything else. Boot times compared to a spinning-disk-only laptop were genuinely shocking.

Lessons learned

  • Always route cables before fitting the motherboard
  • Thermal paste — less is more
  • Full tower cases are magnificent and completely unnecessary unless you're watercooling
  • Planning the storage split from the start saves reinstalling the OS later
  • Check RAM compatibility with the motherboard before buying

Legacy

This machine served as my primary PC for several years before being upgraded. It's where I spent countless hours gaming, got comfortable with Windows internals, and learned to diagnose my first hardware faults — a solid foundation for everything that came after.
