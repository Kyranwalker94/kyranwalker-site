Lockdown Quiz 🎯

A family quiz game built in Scratch during lockdown — with specialist categories, score tracking, and Twitch streaming support.

About the project

When lockdown started I wanted to keep the family connected and having fun, so I built this quiz game in Scratch. The format is designed for a quizmaster's assistant who streams the quiz live on Twitch — questions go out via WhatsApp, players answer, and then the results are entered into the game before the next question is released. The game tallies scores automatically and reads out questions.

There are 6 categories and players pick a specialist category before the game starts. Correct answers in specialist categories earn a bonus 5 points — but a wrong answer costs 5 points.
